#### 功能和特性
- 基于socket实现的c/s架构的的通信
- 服务器和客户心跳连接
- gson实现的消息通信机制
- 注册及登录
- 支持私聊和群聊。
- 动态更新用户列表以及用户消息提示
- 支持emoji表情,以及emoji表情选择器
- 服务器端数据库用户记录
- ~~实现文件传输~~
- ~~文件记录~~

#### 使用
1.无需额外配置，已连接公网数据库
2.运行Client目录下MainApp
3.hostname输入1.12.249.224
4.（用户名，密码）：用户1（123，123），用户2（456，456），用户3（test，test）
